About —

With jquery-keyframeAnimation, you can write CSS3 keyframes animation easily. And to replace FLASH, those animation designation use millisecond.

jquery-keyframeAnimation is easy to designation, usefull for desingners. DOM-object is set to last keyframe animation position, not first position. (normally CSS3-keyframes-animation is set to first position! And sometimes it is confused desingners. ).
